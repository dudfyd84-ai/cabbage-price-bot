# AGENTS.md — 코딩 에이전트 공통 규칙

이 프로젝트의 규칙은 도구와 무관하게 동일하다. 어떤 AI 코딩 에이전트를 쓰든
(Claude Code / Gemini · Antigravity / 기타) 아래 문서를 먼저 읽고 그 규칙을 따른다.

- **`CLAUDE.md`** — 프로젝트 규칙 원본 (로컬 실행, 협업 규칙, 파일 소유, 손대지 말 것, 코딩 원칙).
- **`ROADMAP.md`** — 트랙 배정과 태스크.
- **`CONTRIBUTING.md`** — 브랜치·PR 상세 흐름.

## 반드시 지킬 것 (요약)
- `main` 직접 push 금지. `feature/트랙-기능` 브랜치 → PR → 리뷰 → 병합.
- PR 본문에 `Closes #이슈번호`.
- 데이터·모델 파일(`*.csv`, `model_*.pkl`, `accuracy.json`)과 비밀키는 건드리지 않는다.
- 본인 트랙 파일 위주로만 수정한다.
- 계획을 먼저 세워 사람이 승인한 뒤 구현한다. 한국어·최소 코드.
